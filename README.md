# Student Data Analysis

A Python console program for cleaning, sorting, and visualizing student scores using *NumPy, **Pandas, and **Matplotlib*.  
This project demonstrates beginner-friendly data analysis techniques in Python, including handling missing values, sorting data, calculating averages, and creating visualizations.

## Features

- Clean missing data using mean imputation  
- Sort students by scores in any subject  
- Calculate average scores using NumPy  
- Visualize student scores with Matplotlib bar charts  
- Interactive console-based program  

## How to Run

1. Save student_scores_analysis.py on your computer.  
2. Open *Command Prompt* (Windows) or *Terminal* (Mac/Linux) and navigate to the folder containing the file.  
3. Run the program:

```bash
python student_scores_analysis.py
